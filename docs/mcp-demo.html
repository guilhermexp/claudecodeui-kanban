üõ†Ô∏è Adding a Server Step-by-Step

1. Click "Add MCP Server"

Opens the server configuration form

2. Fill in Server Details

  • Name: Unique identifier (e.g., "filesystem")
  • Type: stdio, http, or sse
  • Command: The command to run (e.g., "npx")
  • Arguments: Command arguments (click + to add)
  • Environment: Environment variables (click + to add)

3. Test Connection

Click "Test" to verify the server works before saving

4. Save and Enable

Save the server, then enable its tools in Settings > Tools
