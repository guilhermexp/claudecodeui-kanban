# Changelog Directory

This directory maintains the historical record of all changes to the Claude Code UI project.

## Files

- **CHANGELOG_FORK.md** - Main changelog following semantic versioning
  - Documents all releases and major changes
  - Organized by version and change type (Added, Changed, Fixed, Removed)
  - Links to related commits

## Organization Guidelines

- One changelog file per major version or development phase
- Use semantic versioning (MAJOR.MINOR.PATCH)
- Group changes by type: Added, Changed, Fixed, Removed, Deprecated
- Include commit hashes for easy reference
- Link to related issues and documentation
